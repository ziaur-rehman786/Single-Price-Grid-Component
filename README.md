# Single Price Grid Component

A responsive pricing card component built with HTML and CSS. This component features a clean design with three sections: community introduction, monthly subscription details, and a benefits list.

## Features

- Fully responsive design (mobile & desktop)
- Hover effects on interactive elements
- Clean, modern UI matching the provided design specifications
- Uses Karla font family
- Semantic HTML structure

## Technologies Used

- HTML5
- CSS3 (Grid & Flexbox)
- Google Fonts (Karla)

## Preview

Check out the `preview.jpg` to see how it looks, or just open `index.html` in your browser.

## Usage

Simply open `index.html` in your web browser to view the component. No build process or dependencies required.
